This is an organizer for pulling and storing stack calls from CE. You just copy the full line from a stack call and it will store it in a file in MyPC>Documents

It also lets you copy what it stored for easy use in AA. It's Mainly to help keep me organized
and to help with not having to backtrack because I didn't write something down, but if it helps someone else, that's great

The function Name line is so you can Identify what the caller is for and the program also lets you sort the stack calls by game so that it's not cluttered
